# Documentation

## Command in Ngejawi

|  NGEJAWI |  MEAN           |
| -------- |  -------------  |
| LAN      |  AND            |
| UTAWI    |  OR             |
| SANES    |  NOT            |
| YEN      |  IF             |
| LIYANYEN |  ELSEIF  	     |
| LIYAN    |  ELSE   	     |
| NGGO     |  FOR    	     |
| KANTHI   |  TO             |
| STEP     |  INCREMENT FOR  |
| NALIKA   |  WHILE          | 
| DADOS    |  THEN           |

## Examples

## Variabel
**example:**
```
ngejawi > VAR a=3
3
ngejawi > VAR a=5
5
ngejawi > VAR c=a+b
8
ngejawi > c==b
0
ngejawi > c!=b
1
ngejawi > c<b
0
ngejawi > c>b
1
```
## Operation Math
**example:**
```
ngejawi > 2 + 4
6
ngejawi > 8-28
-20
ngejawi > 8/5
1.6
ngejawi > 3*5
15
ngejawi > (3+8)-7*(1+1)
-3
ngejawi > --7
7
ngejawi > -+7
-7
ngejawi > 2^8
256
ngejawi > 256^(1/8)
2.0
```
